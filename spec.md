# Personal Website — Andrew Ju

## Overview
Clean, minimal single-page portfolio built with Next.js + Tailwind CSS. Light theme. Showcases education, experience, research, projects, and contact info.

## Sections (scroll order)
1. **Hero** — Name, tagline ("Math of Computation @ UCLA"), social links (LinkedIn, GitHub, email)
2. **About** — Brief intro combining quant finance + ML + full-stack background
3. **Experience** — Timeline cards: Jiyu Fund, One Step Ahead, AdaptiConnect, FS-CFTRA, Yale Townsend Lab
4. **Research & Publications** — Papers with venues (AsianFA, AACR, Science Journal of Public Health)
5. **Projects** — Card grid: Bruin Plans, Multichat, Stock Prediction Engine, Market Screener, Music Classification
6. **Skills** — Languages + tools in tag/pill format
7. **Contact** — Email, LinkedIn, GitHub links

## Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS
- TypeScript
- Deploy target: Vercel or GitHub Pages

## Design
- Light theme, clean sans-serif typography (Inter)
- Responsive (mobile-first)
- Subtle scroll animations (CSS only, no heavy libs)
- No dark mode toggle (light only)

## Definition of Done
- [ ] All sections render with real resume data
- [ ] Fully responsive (mobile, tablet, desktop)
- [ ] Accessible (semantic HTML, alt text, contrast)
- [ ] Fast load (no unnecessary dependencies)
- [ ] Deployable with `npm run build`
